# Dice Roller App

An Android app built with Kotlin and Jetpack Compose that simulates rolling a dice.

## Features
- Random dice roll (1-6)
- Material Design UI
- Interactive button
- Vector drawable dice images

## How to Run
1. Clone this repository
2. Open in Android Studio
3. Run on Android device or emulator

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
